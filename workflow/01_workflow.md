## Setting up your workspace


!SLIDE commandline

    $ git clone git@github.com:username/rails.git
    $ cd rails
    $ git remote add upstream git://github.com/rails/rails.git

    $ git checkout -b my_contribution

    $ git fetch upstream
    $ git merge upstream/master
